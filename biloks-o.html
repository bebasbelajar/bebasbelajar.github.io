15. Biloks H=1,O=-2 tentukan biloks Fe dalam FeH2O3


Bilangan oksidasi unsur Fe dalam senyawa FeH₂O₃ adalah +4.

Pembahasan :Biloks Fe dalam senyawa FeH₂O₃

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks FeH₂O₃ = 0

biloks Fe + (2 × biloks H) + (3 × biloks O) = 0

biloks Fe + (2 × (+1)) + (3 × (-2)) = 0

biloks Fe + (+2) + (-6) = 0

biloks Fe + (-4) = 0

biloks Fe = +4

⇒ Kesimpulan, maka bilangan oksidasi unsur Fe dalam senyawa FeH₂O₃ adalah +4.

16. tentukan biloks dari atom o dalam h2o2


Bilangan oksidasi  O ( oksigen) dalam [tex]H_{2}O_{2}[/tex] adalah -1.  [tex]H_{2}O_{2}[/tex] adalah senyawa  contoh senyawa peroksida.

Pembahasan

∴ Bilangan oksidasi O dalam [tex]H_{2}O_{2}[/tex]  adalah -1 sesuai aturan biloks no 9

Pembuktian biloks O adalah -1

∴ Biloks O pada [tex]H_{2}O_{2}[/tex]  adalah -1.

Diperoleh dari subtitusi biloks yang sudah diketahui pada [tex]H_{2}O_{2}[/tex]  untuk memperoleh biloks O

∵Biloks total [tex]H_{2}O_{2}[/tex]  adalah 0 ( aturan biloks 2) dan Biloks H adalah +1 (aturan biloks ke 8)

   Biloks [tex]H_{2}O_{2}[/tex] = 0

  (jumlah atom H x biloks atom H) + (jumlah atom O x biloks atom O) = 0

  (2 x biloks O) + ( 2 x (+1)) = 0

  2 Biloks O + 2 = 0

  2 Biloks O = 2- 0

  Biloks O = [tex]\frac{-2}{2}[/tex]

  Biloks O = -1

26. 1. Biloks Mn pada K2MnO4 2. Biloks N pada NH4+ 3. Biloks Cr pada K2CrO7 4. Biloks O pada Na2O2 5. BILOKS Mn pada KMnO4 6. Biloks S pada Na2SO4 7. Biloks H pada Na2H2


1. Biloks Mn pada K₂MnO₄ adalah +6.

2. Biloks N pada NH₄⁺ adalah -3.

3. Biloks Cr pada K₂CrO₇ adalah +12.

4. Biloks O pada Na₂O₂ adalah -1.

5. Biloks Mn pada KMnO₄ adalah +7.

6. Biloks S pada Na₂SO₄ adalah +6.

7. Biloks H pada Na₂H₂ adalah -1.

Pembahasan : 1. Biloks Mn pada K₂MnO₄

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂MnO₄ = 0  

(2 × biloks K) + biloks Mn + (4 × biloks O) = 0  

(2 × (+1)) + biloks Mn + (4 × (-2)) = 0  

(+2) + biloks Mn + (-8) = 0  

biloks Mn + (-6) = 0  

biloks Mn = +6

2. Biloks N pada NH₄⁺

Biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H) = +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N = (+1) - 4

biloks N = -3

3. Biloks Cr pada K₂CrO₇

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0  

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0  

(+2) + biloks Cr + (-14) = 0  

biloks Cr + (-12) = 0  

biloks Cr = +12

4. Biloks O pada Na₂O₂

biloks Na = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks Na₂O₂ = 0  

(2 × biloks Na) + (2 × biloks O) = 0  

(2 × (+1)) + (2 × biloks O) = 0  

(+2) + (2 × biloks O) = 0  

(2 × biloks O) = -2

biloks O = [tex]\frac{-2}{2}[/tex]

biloks O = -1

5. Biloks Mn pada KMnO₄

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks KMnO₄ = 0  

biloks K + biloks Mn + (4 × biloks O) = 0  

(+1) + biloks Mn + (4 × (-2)) = 0  

(+1) + biloks Mn + (-8) = 0  

biloks Mn + (-7) = 0

6. Biloks S pada Na₂SO₄

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

1. Unsur bebas, artinya unsur yg dalam reaksi kimia dalam bentuk atom tunggal atau dwi atom atau tetra atom atau okta atom selalu mempunyai bilangan oksidasi = 0, contohnya Na, Mg, Fe, Mn, H2, O2, Cl2, P4, S8

2. Atom H dalam senyawanya bila berikatan dg unsur non logam selalu mempunyai bilangan oksidasi = +1, contoh dalam senyawa H2O, HF, HCl. Tetapi bila dalam senyawa yg berikatan dengan unsur logam, akan mempunyai bilangan oksidasi = -1, contohnya dalam senyawa NaH, CaH2,  

3. Bilangan oksidasi O dalam senyawanya selalu -2, contohnya dalam CO2, K2O, MgO, tetapi dalam senyawa :

a. Biner flourida bilangan oksidasi O = +2, contoh OF2

b. Peroksida bilangan oksidasi O = -1, contoh H2O2, Na2O2

c. Superokdida bilangan oksidasi O = -1/2, misalnya dalam senyawa KO2, CsO2

5. Bilangan oksidasi logam golongan IIA (Be, Mg, Ca, Sr, Ba) selalu +2, contoh BaO, bilangan oksidasi Ba= +2

5.bilangan oksidasi unsur transisi (Fe, Sc,Ti,V,Cr,Mn,Fe,Co, Ni, Cu) mempunyai lebih dari satu bilangan oksidasi dan tergantung dalam senyawanya, contoh FCl2, biloks Fe =+2, tetapi dlm senyawa FeCl3 biloks Fe=+3

6. Bilangan oksidasi unsur non logam (C, N,F,Cl,Br,I,P,S) bila dalam senyawa : a. Biner yg terdiri dari logam dan non logam maka biloks nya = -1, contoh dlm senyawa KCl, biloks Cl =-1

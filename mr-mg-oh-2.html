Massa Atom Relatif (Ar)

Massa atom relatif adalah massa suatu atom yang ditentukan dengan cara membandingkan dengan massa atom standar. Massa atom relatif, disingkat dengan Ar. Berdasarkan IUPAC (International Union of Pure and Applied Chemistry) yang digunakan sebagai standar penentuan massa atom relatif adalah atom C-12. Kemudian didapatkan rumus penentuan massa arom relatif (Ar) adalah perbandingan massa rata – rata unsur dengan 1/12  massa 1 atom C-12.

Ar X = [tex]\frac{massa~1~atom}{\frac{1}{12}~x~massa~1~atom~C-12 }[/tex]

dengan:

Ar X = massa atom relatif X

Massa 1 atom C-12 = 12 sma

Massa rata – rata atom X merupakan massa rata – rata dari semua isotop X yang ada di alam. Massa atom relatif tidak mempunyai satuan.

Massa atom suatu unsur dapat juga ditentukan dengan mengguunakan massa isotop dan kelimpahan dari masing-masing isotop yang berada di alam.

Contoh

Massa rata-rata 1 atom unsur X adalah 4,037 x 10⁻²³ gram,sedangkan massa 1 atom C-12 adalah 1,99268 x 10⁻²³ gram.Berapakah massa atom relatif unsur X?

Ar X = [tex]\frac{4,037.10^{-23} }{\frac{1}{12}~x~1,99268.10^{-23} }[/tex]

Ar X = [tex]\frac{4,037.10^{-23} }{0,166.10^{-23}}[/tex]

Ar X = 24,319


Contoh penentuan massa atom relatif dari persentase/kadar isotop

Unsur klorin terdiri atas 75,53% isotop Cl-35 dan 24,47% isotop Cl-37. Massa Cl-35 adalah 34,969  sma sedangkan Cl-37 adalah 36,965 sma. Berapakah massa atom relatif Cl?

Jawab :

Jawab :

Ar Cl = [tex]\frac{75,53}{100}[/tex] x 34,969 + [tex]\frac{24,47}{100}[/tex] x 36,965

Ar Cl = 26,41 + 9,05

Ar Cl = 35,46


Massa Molekul Relatif (Mr)

Massa molekul relatif  (Mr) adalah jumlah total dari massa atom relatif unsur-unsur penyusunnya dalam suatu senyawa atau molekul. Massa molekul relatif (Mr) digunakan untuk sneyawa atau molekul sedangkan massa atom relatif digunakan untuk unsur atau atom. Rumus untuk menentukan massa molekul relatif (Mr) adalah:

Mr AnBm = n. Ar A + m. Ar B

Contoh :

Massa atom relatif setiap unsur penyusun dari senyawa dapat dikat lihat pada tabel sistem periodik unsur.

Ar H = 1

Ar N = 14

Ar O = 16

Ar C = 12

Ar S = 32

Mr HCl = 36,5

b. NH₃

Mr NH₃ = 1. Ar N + 3. Ar H

Mr NH₃ = 1. 14 + 3. 1

Mr NH₃ = 17

c. CH₃COOH

Mr CH₃COOH = 2. Ar C + 4. Ar H + 2. Ar O

Mr CH₃COOH = 2. 12 + 4. 1 + 2. 16

Mr CH₃COOH = 24 + 4 + 32

Mr CH₃COOH = 60

d. BaSO₄

Mr BaSO₄ = 1. Ar Ba + 1. Ar S + 4. Ar O

Mr BaSO₄ = 1. 137 + 1. 32 + 4. 16

Mr BaSO₄ = 137 + 32 + 64

Mr BaSO₄ = 233

e. Ca(OH)₂

Mr Ca(OH)₂ = 1. Ar Ca + 2. Ar O + 2. Ar H

Mr Ca(OH)₂ = 1. 40 + 2. 16 + 2. 1

Mr Ca(OH)₂ = 40 + 32 + 2

Mr Ca(OH)₂ = 74

Naskah Soal:
Kelarutan Mg(OH)₂ dalam air pada 25°C adalah 1,16 mg per 100 mL (Mr=58) harga Ksp dari Mg(OH)₂ adalah

Diketahui : Massa Mg(OH)₂ = 1,16 mg =  0.00116 gram
                  Volume Pelarut = 100 mL = 0,1 L                   
                  Mr Mg(OH)₂ = 58
                 
Ditanya :  Ksp Mg(OH)₂ .............?

Jawab:

Untuk Menjawab soal tersebut dengan mudah saya akan mencoba untuk memecahnya dalam beberapa langkah:

Langkah pertama:
Karena kelarutan dari Mg(OH)₂ diketahui dalam satuan massa per volume pelarut maka kita harus mengkonversinya terlebih dahulu menjadi Molaritas (Molar) dengan cara sebagai berikut:
1. Mengkonversi Massa menjadi mol dengan rumus

                              Mol (n) = Massa : Mr                                
dan 

                              Molaritas (M)  =  Mol (n) : Volume Pelarut

Langkah Kedua:
Sehingga, kita bisa merubah massa Mg(OH)₂ menjadi Molaritas dengan mudah dengan cara berikut:
Menghitung mol Mg(OH)₂ = massa : Mr
                                           = 0,00116 : 58                                           
                                           = 2 x 10⁻⁵ mol

Menghitung Molaritas Mg(OH)₂ =  mol : Volume Pelarut                                                 
                                                      = 2 x 10⁻⁵ mol : 0,1 L                      
                       Molaritas Mg(OH)₂ = 2 x 10⁻⁴ M

Langkah Ketiga:
Menentukan nilai Ksp Mg(OH)₂ dengan cara menuliskan terlebih dahulu persamaan reaksi kesetimbangannya, seperti berikut:

                     Mg(OH)₂   <==>   Mg⁺²   +  2 OH⁻

Jika kita umpamakan nilai kelarutan dari Mg(OH)₂ adalah s, maka:

                       Mg(OH)₂   <==>   Mg⁺²   +  2 OH⁻                           
                           s                        s              2s

Sehingga nilai Ksp bisa kita hitung:                    
                       Ksp Mg(OH)₂ = [Mg⁺²] [OH⁻]²                                         
                                              = (s) (2s)²                                         
                                              =  (s) (4s²)                       
                       Ksp Mg(OH)₂ =  4s³

Tinggal kita masukkan nilai s yang kita miliki, sehingga:                         
                      Ksp Mg(OH)₂  =  4 (2x10⁻⁴)³                                             
                                              =  4 ( 8 x 10⁻¹²)                                       
                                              =  32 x 10⁻¹²

jadi, nilai hasil kali kelarutan dari 1,16 gram Mg(OH)₂ dalam 100 mL pelarut adalah 32 x 10⁻¹²

9. sebanyak 0,58 gram Mg(OH)2 yang terlarut dalam 2 liter larutan (Mr Mg(OH)2 = 58 ) mempunyai pH sebesar...


* Mencari mol
mol = massa/Mr
mol = 0,58/58
mol = 0,01 mol

* Mencari Molaritas
M = mol/V
M = 0,01/2
M = 5 x 10⁻³ M

* Mencari [OH⁻]
[OH⁻] = M . Valensi basa
[OH⁻] = 5 x 10⁻³ . 2
[OH⁻] = 10⁻²

* Mencari pOH
pOH = -log [OH]
pOH = -log [10⁻²]
pOH = 2

* Mencari pH
pH = 14 - pOH
pH = 14 - 2
pH = 12

Jadi, pH larutan sebesar 12

13. Berapa Mr molekul NaOH,Mg(OH)2,Ca(OH)2,Be(OH)2?

Mr Be(OH)2 = 43

Mr NaOH = Ar Na + Ar O + Ar H

=> 23 + 16 + 1

=> 40

Mr Mg(OH)2 = Ar Mg + ( 2 x Ar O ) + ( 2 x Ar H )

=> 24 + ( 2 x 16 ) + ( 2 x 1 )

=> 24 + 32 + 2

=> 58

Mr Ca(OH)2 = Ar Ca + ( 2 x Ar O ) + ( 2 x Ar H )

=> 40 + ( 2 x 16 ) + ( 2 x 1 )

=> 40 + 32 + 2

=> 74

Mr Be(OH)2 = Ar Be + ( 2 x Ar O ) + ( 2 x Ar H )

=> 9 + ( 2 x 16 ) + ( 2 x 1 )

=> 9 + 32 + 2

=> 43

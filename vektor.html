Panjang proyeksi vektor B + Vektor C pada vektor A adalah [tex]|\overrightarrow{(\vec{B}+\vec{C})_A}| = \frac{\sum \limits^{}_{n=x,y,z}(4n_i+n_j+n_k)(n_i-n_j+n_k)}{\sqrt{(\hat{A_x})^2+(\hat{A_y})^2+(\hat{A_z})^2}}[/tex]

PENDAHULUAN

Vektor secara umum didefinisikan sebagai suatu besaran yang memiliki arah. Dalam Matematika, vektor merupakan suatu garis yang memilih arah. Vektor dapat dinyatakan dalam 2 bentuk yaitu bentuk garis dan bentuk aljabar.

[tex]\textbf{Vektor sebagai sebuah garis}. \\\overrightarrow{AB} = \overrightarrow{B} - \overrightarrow{A} \\\\\textbf{Vektor sebagai sebuah aljabar} \\Vektor ~2 ~dimensi : r = x\hat{i} + y\hat{j} \\Vektor ~3 ~dimensi : r = x\hat{i} + y\hat{j} + z\hat{k} \\[/tex]

Suatu vektor juga memiliki nilai besaran / panjang yang dapat dinyatakan dalam bentuk tanda mutlak berikut.

[tex]Vektor ~pada ~bidang : |r| = \sqrt{x^2 +y^2} \\Vektor ~pada ~ruang : |r| = \sqrt{x^2 +y^2 +z^2} \\[/tex]

Perihal lain yang terkait pada vektor adalah suatu proyeksi orthogonal antar vektor.

[tex]Panjang~proyeksi ~vektor ~a ~pada ~b : a_b = \frac{a \cdot b}{|b|}\\Panjang~proyeksi ~vektor ~b ~pada ~a : b_a = \frac{a \cdot b}{|a|} \\[/tex]

[tex]proyeksi ~vektor ~a ~pada ~b : a_b = \frac{a \cdot b}{|b|^2}\vec{b} \\proyeksi ~vektor ~b ~pada ~a : b_a = \frac{a \cdot b}{|a|^2}\vec{a} \\[/tex]

11. jika p vektor Q vektor R vektor vektor berturut-turut merupakan vektor posisi dari titik sudut jajargenjang pqrs dengan PQ vektor dan vektor maka X vektor adalah ​


Jika p, q, r, s berturut - turut adalah vektor posisi titik - titik sudut jajaran genjang PQRS dengan PQ sejajar SR, maka s = ....

Pembahasan :

Jajar genjang memiliki dua pasang sisi yang sejajar dan sama panjang

Dua buah vektor dikatakan sama jika panjangnya sama dan arahnya juga sama.

Jadi pada jajar genjang PQRS, dua buah vektor yang sama adalah

PQ = SR atau PS = QR

PQ = SR

q - p = r - s

s = p + r - q

PS = QR

s - p = r - q

s = p + r - q

Jadi vektor s pada jajar genjang PQRS adalah

s = p + r - q

18. jika diketahui vektor oa=P dan vektor ob=q, maka vektor be+vektor cb+ vektor fc + vektor ef + vektor bd + vektor ab =​


BE + CB + FC + EF + BD + AB adalah -2p.

PembahasanVektor-vektor yang searah dan sama panjang besarnya sama, sehingga OA = DO = EF = CB = p, dan OB = EO = FA = DC = q.Hasil penjumlahan vektor adalah garis yang ditarik dari titik pangkal vektor pertama sampai pada titik ujung vektor kedua, sehingga AB = AO + OB.Vektor-vektor yang berlawanan arah akan berlawanan tanda pula. Misalnya, OA = -OA = AO.

Penyelesaian Soal

Diketahui:

OA = p

OB = q

Ditanya:

BE + CB + FC + EF + BD + AB = ...

Penyelesaian:

BE + CB + FC + EF + BD + AB

= 2(BO) + OA + 2(AB) + OA + (BO + OD) + AB

= 2(BO) + OA + 2(AO + OB) + OA + (BO + AO) + (AO + OB)

= 2(-q) + p + 2(-p + q) + p + (-q - p) + (-p + q)

= -2q + p - 2p + 2q + p - q - p - p + q

= (p - 2p + p - p - p) + (-2q + 2q - q + q)

= -2p + 0

= -2p

Jadi BE + CB + FC + EF + BD + AB = -2p.

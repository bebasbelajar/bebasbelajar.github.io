6. rumus kecepatan gerak benda pada peristiwa jatuh bebas please tolong ​


Jawaban:

Secara matematis, gerak jatuh bebas ditulis:

vt = v0 + a.t

Karena v0 = 0 dan a = g, maka rumus di atas berubah menjadi:

vt = g.t

keterangan : vt = kecepatan (m/s), g = percepatan gravitasi (m/s2), dan t = waktu (s).

Adapun rumus untuk mencari suatu ketinggian benda (h) bisa mengganti persamaan gerak lurus berubah beraturan, sehingga di dapatkan persamaan ketinggian benda untuk gerak jatuh bebas yakni:

h = ½ g.t2

Untuk menentukan kecepatan benda yang jatuh bebas dari ketinggian h, dapat ditentukan dengan menggunakan rumus:

vt 2 = 2gh

10. Tuliskan 3 rumusan dasar dalam GLBB dan perubahan nya menjadi rumusan untukGerak Vertikal ke bawah (GVB)Gerak Vertikal ke atas (GVA)Gerak jatuh bebas (GJB)​


Pembahasan

Gerak Lurus Berubah Beraturan (GLBB) adalah gerak pada sistem yang kecepatannya berubah setiap waktu akibat adanya percepatan.

Persamaan GLBB:

[tex]\boxed{\boxed{a = \frac{\triangle v}{t} = \frac {dv} {dt}}}[/tex]

Adanya percepatan menandakan GLBB.

Beberapa persamaan dengan menggunakan metode integral:

[tex]a = \frac{dv}{dt}[/tex]

[tex]dv = a \,dt[/tex]

[tex]\int\limits^{v_t}_{v_o} {} \, dv = \int {a} \, dt[/tex]

[tex]v_t - v_o = a\,.\,t\\v_t = v_o + at[/tex]

[tex]\boxed{\boxed{v_t = v_o + at}}[/tex]

Kita juga dapat membuat persamaan perpindahan posisi.

[tex]v_t = \frac{ds}{dt}[/tex]

[tex]ds = v_t \,dt[/tex]

[tex]\int\limits^{s}_{s_o} {} \, ds =\int (v_o + at)\, dt[/tex]

[tex]s-s_o = v_o\,t + \frac12 \,at^2[/tex]

[tex]\boxed{\boxed{s=s_o +v_o t+\frac12 at^2}}[/tex]

Inilah persamaan hasil dari Integral. Kita dapat menghasilkan persamaan yang terkait dari fungsi waktu menjadi fungsi perpindahan posisi.

[tex]v_t -v_o=at[/tex]

[tex]t = \frac{v_t - v_o}{a}[/tex]

[tex]s = s_o +v_o t + \frac 12 at^2[/tex]

[tex]s = s_o + v_o\frac{v_t - v_o}{a} +\frac12 a(\frac{v_t - v_o}{a})^2[/tex]

[tex]s = s_o + \frac{v_ov_t -v_o\,^2}{a} +\frac{(v_t -v_o)^2}{2a}[/tex]

[tex]s = s_o + \frac{2v_tv_o -2v_o\,^2 + v_t\,^2 - 2v_tv_o +v_o\,^2}{2a}[/tex]

[tex]\boxed{\boxed{s = s_o +\frac{v_t\,^2 -v_o\,^2}{2a}}}[/tex]

Contoh kasus nyata GLBB adalah:

Gerak Vertikal ke bawah (GVB)Gerak Vertikal ke atas (GVA)Gerak jatuh bebas (GJB)

Ketiga kasus ini, hal yang menyebabkan gerakan tersebut adalah GLBB adanya percepatan gravitasi.

[tex]a = \pm g[/tex]

perpindahan posisi di sini adalah, tentu, ketinggian

[tex]s = h[/tex]

Analisis GVB

Pada GVB, Benda bergerak searah dengan gravitasi sehinga:

[tex]a = g[/tex]

Berikut perhitugan persamaan:

[tex]v_t = v_o + at\\v_t = v_o +(g)t\\\\\boxed{\boxed{v_t = v_o + gt}}[/tex]

[tex]s = s_o + v_ot + \frac12 at^2\\h = h_o + v_o t + \frac 12 (g)t^2[/tex]

[tex]\boxed{\boxed{h = h_o +v_o t + \frac 12 gt^2}}[/tex]

[tex]s = s_o + \frac{v_t\,^2 - v_o \,^2}{2a}[/tex]

[tex]h = h_o + \frac{v_t\,^2 - v_o\,^2}{2g}[/tex]

[tex]\boxed{\boxed{h = h_o + \frac{v_t\,^2 - v_o\,^2}{2g}}}[/tex]

Analisis GVA

Pada GVA, arah awal benda berlawanan arah dengan gravitasi, sehingga:

[tex]a = -g[/tex]

Berikut perhitugan persamaan:

[tex]v_t = v_o + at\\v_t = v_o +(-g)t\\\\\boxed{\boxed{v_t = v_o - gt}}[/tex]

[tex]s = s_o + v_ot + \frac12 at^2\\h = h_o + v_o t + \frac 12 (-g)t^2[/tex]

[tex]\boxed{\boxed{h = h_o +v_o t - \frac 12 gt^2}}[/tex]

[tex]s = s_o + \frac{v_t\,^2 - v_o \,^2}{2a}[/tex]

[tex]h = h_o + \frac{v_t\,^2 - v_o^2}{-2g}[/tex]

[tex]\boxed{\boxed{h = h_o + \frac{v_o\,^2 - v_t\,^2}{2g}}}[/tex]

Analisis GJB

GJB merupakan GVB tanpa kecepatan awal. Jadi:

[tex]v_o = 0\\a = g[/tex]

Berikut perhitugan persamaan:

[tex]v_t = v_o + at\\v_t =0+(g)t\\\\\boxed{\boxed{v_t = gt}}[/tex]

[tex]s = s_o + v_ot + \frac12 at^2\\h = h_o + 0\,.\, t + \frac 12 (g)t^2[/tex]

[tex]\boxed{\boxed{h = h_o + \frac 12 gt^2}}[/tex]

[tex]s = s_o + \frac{v_t\,^2 - v_o \,^2}{2a}[/tex]

[tex]h = h_o + \frac{v_t\,^2 -0^2}{2g}[/tex]

[tex]\boxed{\boxed{h = h_o + \frac{v_t\,^2}{2g}}}[/tex]

Nama-nama variabel:

[tex]a[/tex] : Percepatan(m/s²)

[tex]v_t[/tex] : kecepatan akhir (m/s)

[tex]v_o[/tex] : kecepatan awal (m/s)

[tex]t[/tex] : lama waktu sistem bergerak (s)

[tex]s[/tex] : posisi objek (m)

[tex]s_o[/tex] : posisi awal objek (m)

[tex]g[/tex] : percepatan gravitasi (10 atau 9,8 m/s²)

[tex]h[/tex] : posisi ketinggian (m)

[tex]h_o[/tex] : posisi ketinggian awal (m)

vt = v0 + a.t

vt 2 = 2gh

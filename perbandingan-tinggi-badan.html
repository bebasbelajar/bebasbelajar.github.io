13. tinggi badan Laras banding Riska adalah 7 banding 8 tinggi badan Reza banding laras adalah 4 banding 3 Jumlah tinggi badan ketiga anak tersebut 365 cm Berapa cm tinggi badan Laras?


Jawaban:

105 cm

Penjelasan dengan langkah-langkah:

Diketahui : Tinggi badan Laras : Riska

7 : 8

Tinggi badan Reza : Laras

4 : 3

Jumlah Tinggi badan ketiga anak tersebut 365 cm

Ditanya : Berapa cm tinggi badan Laras ?

Jawab :

7/8 = 4/3

8/7 = 4/3

24/21 = 28/21

Riska : Laras : Reza

24 : 21 : 28

24 + 21 + 28 = 73

Tinggi Badan Laras :

21/73 x 365 = 105 cm

Laras = L

Riska = R

Reza = A

Diketahui :

L : R = 7x : 8x

A : L = 4x : 3x

L + R + A = 365 cm

Ditanya :

L = ?

Jawab :

Kita Samakan Perbandingan Atas Dengan Bawah.

Karena Laras ada di atas dan dibawah maka

L : R : A = ( 7 × 3 )x : ( 8 × 3 )x : ( 4 × 7 )x

L : R : A = 21x : 24x : 28x

L + R + A = 365 cm

21x + 24x + 28x = 365 cm

73x = 365 cm

x = 5 cm

Laras :

21x

= 21 × 5 cm

= 105 cm

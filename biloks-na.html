4. Pada reaksi 2Na+2H2O→2NaOH+H2 maka biloks Na akan berubah dari?


Dalam persamaan reaksi: 2Na + 2H₂O → 2NaOH + H₂, maka bilangan oksidasi Na akan berubah dari 0 menjadi +1.

Pembahasan :

2Na + 2H₂O → 2NaOH + H₂

Tahap 1 :

Tentukan terlebih dahulu biloks Na :  

biloks Na = 0

aturan biloks :  

Bilangan oksidasi unsur bebas adalah nol. Contoh : Bilangan oksidasi atom-atom pada C, Ne, H₂, O₂, Cl₂ , P₄, S₈, Cu, Fe, Na adalah nol.

Tahap 2 :

Selanjutnya tentukan biloks Na dalam NaOH :  

Biloks Na dalam NaOH

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NaOH = 0

biloks Na + biloks O + biloks H = 0  

biloks Na + (-2) + (+1) = 0

biloks Na + (-1) = 0  

biloks Na = +1

Reaksi perubahan bilangan oksidasi :

2Na + 2H₂O → 2NaOH + H₂

   |___________|  

   0                      +1  

Unsur Na mengalami perubahan biloks dari 0 menjadi +1.

9. Hitunglah biloks pada reaksi h² so⁴ + na oh h²o + na² so⁴


Jawaban:

 H2SO4    +    NaOH    ⇒    H2O    +    Na2SO4

+1 | +6 | -2     +1 | -2 | +1        +1 | -2         +1 | +6 | -2

-------------------------------------------------------------------------------------------------------------

Penjelasan:

1) Biloks S pada senyawa H2SO4:

H2SO4 = 0

(1 x 2) + S + (-2 x 4) = 0

2 + S - 8 = 0

S = 6

maka, biloks unsur-unsur di H2SO4 adalah:

H = +1; S = +6; O = -2

2) Biloks unsur pada NaOH:

Na = +1; O = -2; H = +1

3) Biloks unsur pada H2O:

H = +1; O = -2

4) Biloks unsur S pada Na2SO4:

Na2SO4 = 0

(1 x 2) + S + (-2 x 4) = 0

2 + S - 8 = 0

S = 6

maka, biloks setiap unsur dalam Na2SO4 adalah:

Na = +1; S = +6; O = -2

jika disusun, seperti ini:

 H2SO4    +    NaOH    ⇒    H2O    +    Na2SO4

+1 | +6 | -2     +1 | -2 | +1        +1 | -2         +1 | +6 | -2

-------------------------------------------------------------------------------------------------------------

Aturan Biloks:

1. Biloks unsur bebas = 0

contoh: Na, Mg, S, P4, O2

2. Biloks dalam senyawa:

Logam utama (IA, IIA, Aluminium) = +nomor golongan

        Contoh: biloks Na pada NaCl = +1 (Na berada pada golongan IA)

Total biloks dalam satu senyawa = 0Biloks O = -2

        Pengecualian: pada senyawa peroksida (O = -1) contoh: H2O2, Na2O2

                                 pada senyawa superoksida (O = -1/2) contoh: NaO2

                                 pada senyawa F2O (O = +2)

Biloks F = -1Biloks ion = muatannya

        Contoh: Biloks S²⁻ = -2; biloks NH4⁺ = +1

Biloks H = +1

        Pengecualian: pada senyawa hibrida (H = -1)

                                 senyawa hibrida = H + logam reaktif (golongan IA, IIA)

                                 contoh senyawa hibrida: NaH, BaH2

12. Tentukan Biloks dari senyawa berikut Na+ NH4+ N dari senhawa NH4 Cr2o7^2-


Kelas: XII SMA
Pelajaran: Kimia
Pembahasan: Aturan Biloks

a) Na^+ = +1
b) NH4^- = -1 karena nilai biloks N = +3 sedangkan H = -1 (gol. IA) maka 4H = -4 sehingga dijumlahkan nilai biloksnya jadi +3 + (-4) = -1

c) nilai biloks atom N dari NH4^- adalah +3 karena
(1 × biloks N) + (4 × biloks H) = -1
biloks N + (4 × (-1)) = -1
biloks N + (-4) = -1
biloks N = -1 + 4 = +3 .

d) Cr3O7^-2 = -2 karena nilai biloks atom Cr = +6 maka 2Cr = +12, nilai biloks O non-peroksida = -2 maka 7O = -14. sehingga +12 + (-14) = -2

e) Nilai biloks atom Cr dari Cr2O7^2 =
(2 × biloks Cr) + (7 × biloks O) = -2
2 × biloks Cr + (7×(-2)) = -2
2 × biloks Cr + (-14) = -2
2 × biloks Cr = -2 + 14
biloks Cr = 12/2 = +6

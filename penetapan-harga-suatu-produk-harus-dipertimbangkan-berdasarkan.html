Setiap perusahaan menginginkan mendapatkan laba untuk setiap produksinya. Laba bisa dihitung dengan mencari titik impas atau BEP dahulu. Karena ada 3 alternatif yang harus dipilih, maka akan dicoba dihitung satu persatu.

Fungsi biaya variabel : VC = 1.050Q + 30.000.000

Fungsi biaya total : TC = FC + VC , dimana

TC = 50.000.000 + 1.050Q = 50.000.000 + 1.050Q

Laba terjadi saat penerimaan total (TR) lebih besar dari biaya total (TC).

Alternatif I dimana harga (P) = Rp 1.650 dan Q' = 20.000 unit.

Fungsi penerimaan total :

TR = P.Q

TR = 1.750Q + 1.650Q'

TR = 1.750(160.000) + 1.650(20.000)

= 280.000.000 + 33.000.000

= 313.000.000

TC = 50.000.000 + 1.050(Q + Q')

= 50.000.000 + 1.050(180.000)

= 239.000.000

Dapat dilihat untuk alternatif I, laba yang diperoleh :

Laba = TR – TC

= 313.000.000 – 239.000.000

= 74.000.000

Alternatif II, harga (P) = Rp 1.475 dan Q' = 40.000 unit.

TR = 1.750Q + 1.475Q'

= 1.750(160.000) + 1.475(40.000)

= 280.000.000 + 59.000.000

= 339.000.000

TC = 50.000.000 + 1.050(Q + Q')

= 50.000.000 + 1.050(200.000)

= 260.000.000

Laba = 339.000.000 – 260.000.000

= 79.000.000

= 1.750(160.000) + 1.600(75.000)

26. Tuliskan dan setarakan persamaan reaksi antara besi (III) oksida dengan karbon monoksida membentuk logam besi dan karbon dioksida​.


Fe2O3 + 3CO -----> 2Fe + 3CO2. Reaksi antara besi (III) oksida dengan karbon monoksida membentuk logam besi dan karbon dioksida sudah setara.

Pembahasan:

Fe2O3 + CO -----> Fe + CO2

Menghitung nilai konstanta

aFe2O3 + bCO -----> cFe + dCO2

2a = c

3a + b  = 2d

b = d

Misal a = 1, maka:

2a  = c

2 (1) = c = 2

3a + b  = 2d

3a +  d = 2d

3a = d

3(1) = d = 3

b = d = 3

Maka persamaan reaksinya menjadi

Fe2O3 + 3CO -----> 2Fe + 3CO2

27. Rumus besi (II) oksida dan tembaga (II) oksida berturut-turut yaitu...​


rumus Besi (II) Oksida = FeOrumus Tembaga (II) Oksida = CuO

Pembahasan ☆

⇔ Tata Nama Senyawa

[tex]{\boxed{\sf ❁ Senyawa biner logam + nonlogam}}[/tex]

➸ Logam bernilai (+) atau kation

➸ Nonlogam bernilai (-) atau Anion

➸ Logam yang memiliki muatan lebih dari satu, maka muatan tersebut dinyatakan dengan angka romawi.

Contoh :

Emas ⇔ Au³^+ = Emas (III)

Timah ⇔ Sn⁴^+ = Timah(IV)

➸ Nonlogam senyawanya biasanya berakhiran -Ida dan ber ion negatif

Contoh :

Klorin ⇔ Cl^{-} = Klorida

﹄Contoh Soal :

Magnesium Fluorida

[tex]{\sf Kation \: Logam \: (Magnesium) = Mg^{2+}}[/tex]

Nama kation = Magnesium

[tex]{\sf Anion \: Nonlogam \: (Fluorida) = F^{-}}[/tex]

Nama Anion = Fluorida

Karena belum netral maka pada Fluor diubah menjadi 2F

Masukkan ke dalam rumus

[tex]{\boxed{\blue{\sf A^{x+} + B^{y-} \to A_{y}B_{x}}}}[/tex]

maka,

[tex]{\sf Mg^{2+} + 2F^{-} \to MgF_{2}}[/tex]

Maka penyelesaian soal tersebut sebagai berikut :

[1] Besi (II) Oksida

[tex]{\sf Kation \: Logam \: (Besi) = Fe^{2+}}[/tex]

Nama kation = Besi (II)

[tex]{\sf Anion \: Nonlogam \: (Oksida) = O^{-2}}[/tex]

Nama Anion = Oksida

Sehingga kedua atom tersebut Netral

maka,

[tex]{\boxed{\tt Besi \: (II) \: Oksida = FeO}}[/tex]

[2] Tembaga (II) Oksida

[tex]{\sf Kation \: Logam \: (tembaga) = Cu^{2+}}[/tex]

Nama kation = Cu (II)

[tex]{\sf Anion \: Nonlogam \: (Oksida) = O^{-2}}[/tex]

Nama Anion = Oksida

Sehingga kedua atom tersebut Netral

maka,

[tex]{\boxed{\tt Tembaga \: (II) \: Oksida = CuO}}[/tex]

10. Diberikan IP Address 192.168.69., berapa jumlah host per subnet dari hasil subnetting IP tersebu


Jawaban:

Jumlah Host per Subnet = 2y - 2, dimana y adalah adalah kebalikan dari x yaitu banyaknya binari 0 pada 2 oktet terakhir. Jadi jumlah host per subnet adalah 214 - 2 = 16.382 host. Blok Subnet = 256 - 192 = 64.

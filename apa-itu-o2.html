1. O2 adalah O2 adalah ​


O2 adalah oksigen.

Oksigen merupakan komponen gas yang berfungsi untuk menjaga keberlangsungan hidup manusia


2. diketahui 5,6L O2. ditanya mol O2, massa O2, dan jumlah molekul O2.


Stoikiometri
Kimia X

Diketahui 
Volum O₂ = 5,6 liter 

Ditanyakan jumlah mol, massa, dan jumlah molekul

Penyelesaian

[a] Pada keadaan standar (STP)
Jumlah mol = volum / 22,4
n = 5,6 / 22,4
Jadi n = 0,25 mol

[b] Massa = mol x Mr
Mr O₂ = 32
Massa = 0,25 x 32
Jadi massa = 8 gram

[c] Jumlah molekul = mol x bilangan Avogadro
Jumlah molekul = 0,25 x 6,02 x 10²³
Jadi jumlah molekul = 1,505 x 10²³

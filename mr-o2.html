Pendahuluan

jika jumlah partikel O2 sebanyak 3.0×10^23 dan Mr o2 =32 maka massa O2 tersebut sebesar...

____________________________

Pembahasan

Rumus :

n = jumlah partikel / L

Keterangan :

n = molN = jumlah partikelL = bilangan avogadro = 6,02 .10²³

____________________________

Diketahui :

N = 3,0 . 10²³Mr O2 = 32

Ditanya :

m .. ?

Dijawab :

- Langkah ke 1 Menentukan jumlah partikel

n = jumlah partikel / L

n = 3,0 . 10²³ / 6,02 . 10²³

n = 0,49 mol

n = 0,50 mol ( dibulatkan )

- Langkah ke 2 Menentukan massa

n = m / Mr

m = n . Mr

m = 0,50 . 32

m = 16 gram

jadi, massa nya adlah 16 gram

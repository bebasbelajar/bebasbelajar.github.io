Bilangan Positif ( + )

Bilangan Negatif ( - )

[tex]\tt {\bold {\underline {Penjumlahan\:Bilangan\:Bulat}}}[/tex]

Penjumlahan bilangan bulat dapat dilakukan dengan beberapa cara, antara lain menggunakan pasangan kancing positif ( + ), dan pasangan kancing negatif ( - )

➪Penjumlahan pada kancing ( + ) dan kancing ( - )

Catatan :

Pasangan ( + ) dan ( - ) menyatakan bilangan nol

Penjumlahan bilangan nol tidak berarti, artinya mengubah nilai

( tetap )

[tex]\tt {\bold {\underline{Pengurangan\:Bilangan\:Bulat}}}[/tex]

Pengurangan bilangan bulat juga dapat dilakukan dengan beberapa cara seperti pada penjumlahan bilangan bulat

➪Pengurangan pada kancing ( + ) dan ( - )

Sama seperti pada penjumlahan bilangaj bulat, simbol ( + )untuk bilangan positif ( - ) untuk bilangan negatif. Ingat kembali bahwa pasangan ( + ) dan ( - ) menyatakan bilangan nol

Contoh

5 - 3 = 2

4 - 5 = 4 + 0 - 5 = -1

➪Perkalian Bilangan Bulat

Cara mengerjakan bilangan bulat sama seperti perkalian pada bilangan cacah

3 × 2 = 6 ➪Dikurang 2

2 × 2 = 4 ✎Dikurang 2

1 × 2 = 2 ➪Dikurang 2

0 × 2 = 0 ✎Dikurang 2

-1 × 2 = -2 ➪Dikurang 2

-2 × 2= -4 ✎Dikurang 2

-3 × 2 = -6 ➪Dikurang 2

Hasil perkalian dua bilangan bulat bertanda sama adalah bilangan positif

Bilangan positif dikali bilangan positif sama dengan bilangan positif

Bilangan negatif dikali bilangan negatif sama dengan bilangan positif

Hasil perkalian dua bilangan bulat berbeda tanda adalah bilangan bulat negatif

Bilangan positif dikali bilangan negatif sama dengan bilangan negatif

Bilangan negatif dikali bilangan positif sama dengan bilangan negatif

Catatan Rumus

Positif × Positif = Positif

Negatif × Negatif = Positif

Positif × Negatif = Negatif

Negatif × Positif = Negatif

➪Pembagian Bilangan Bulat

Seperti pada perkalian bilangan bulat, hasil pembagian bilangan bulat dapat ditentukan dengan cara sama

Catatan Rumus

Positif ÷ Positif = Positif

Negatif ÷ Negatif = Positif

Positif ÷ Negatif = Negatif

Negatif ÷ Positif = Negatif

Penjelasan dengan langkah-langkah:

ABCD dan PQRS adalah persegi yang sejajar dan memiliki panjang sisi beruturut turut 12 cm dan 8 cm

a. untuk mencari PR dapat digunakan rumus phitagoras

[tex]PR=\sqrt{PQ^2+QR^2}\\ PR=\sqrt{8^2+8^2}\\PR=\sqrt{2 . 8^2}\\PR = 8\sqrt{2}[/tex]

b. untuk mencari AC, dapat digunakan rumus phitagoras

[tex]AC=\sqrt{AB^2+BC^2}\\ AC=\sqrt{12^2+12^2}\\AC=\sqrt{2 . 12^2}\\AC = 12\sqrt{2}[/tex]

c. jarak ABCD dari R

ambil garis tinggi dari R tegak lurus bidang ABCD, anggap titik itu sebagai X, refleksi x terhadap DC adalah Y dan refleksi x terhadap BC adalah Z

CX = [tex]\sqrt{CY^2+CZ^2}[/tex]

CDSR adalah trapesium sama kaki, sehingga CY dapat di hitung dengan cara membagi 2 hasil dari sisi bawah di kurang sisi atas

[tex]\frac{12-8}{2} =\frac{4}{2} =2[/tex]

CDSR kongruen dengan BCRQ, sehingga CY = CZ = 2

substitusi ke CX

CX = [tex]\sqrt{CY^2+CZ^2}[/tex]

CX = [tex]\sqrt{2^2+2^2}[/tex]

CX = [tex]\sqrt{2. 2^2}[/tex] = [tex]2\sqrt{2}[/tex]

jarak R ke bidang ABCD dapat di hitung

[tex]RX=\sqrt{CX^2+CR^2}\\ RX=\sqrt{(2\sqrt{2})^2+10^2}\\RX=\sqrt{4.2 + 100}\\RX = \sqrt{8+100}\\RX=\sqrt{108}\\ RX=\sqrt{9.4.3} \\RX=3.2\sqrt{3} \\RX = 6\sqrt{3}[/tex]

20. no 12 Kimia Kelas 12


Tf° = 178,4°C
Kf = 37,7 °C kg/mol
massa terlarut = 0,94 gr
m kamfor = 25 gr
Tf = 170,8°C
Mr = ...?

∆Tf = Tf° - Tf = 178,4°C - 170,8°C = 7,6°C

i dari zat terlarut anggap saja 1

∆Tf = Kf × m × i
7,6°C = 37,7°C kg/mol × m × 1
m = 0,2016 m

m = (massa ter/Mr) × (1000/m pel)
0,2016 m = (0,94 gr/Mr) × (1000 gr/25 gr)
Mr = 186,5

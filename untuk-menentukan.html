LISTRIK DINAMIS

Rumus umum Tegangan Listrik :

V = i x R ➡ i = V / R

Rumus Daya Listrik :

P = W / t

P = V x i x t / t

P = V x i

P = i x R x i

P = i² x R

P = V x i

P = V x V / R

P = V² / R

Rumus Tegangan Listrik :

V = i x R

Rumus Tahanan Listrik :

R = V / i

Apabila ada rangkaian tahanan yang disusun secara seri

Rₛ = R₁ + R₂ + R₃ + ... + Rₙ

Apabila ada rangkaian tahanan yang disusun secara paralel

Rₚ = 1/R₁ + 1/R₂ + 1/R₃ + ... + 1/Rₙ

Rumus Kuat Arus Listrik :

i = V / R

Pembahasan

Daya Listrik adalah kemampuan suatu Peralatan Listrik untuk melakukan usaha akibat adanya Perubahan kerja maupun Perubahan muatan Listrik tiap satuan Waktu.

》Pembahasan

Listrik adalah energi, sehingga sesuai dengan hukum kekekalan energi untuk menghasilkan energi listrik perlu adanya alat yang dapat mengubah energi lain menjadi energi listrik. Secara umum sumber arus listrik terdiri dari dua jenis, yaitu sumber arus searah ( Direct Current = DC ) dan sumber arus bolak balik ( Alternating Current = Ac ).

Berdasarkan dapat atau tidaknya diisi ulang, sumber arus listrik dibedakan menjadi elemen primer dan elemen sekunder.

Elemen primer adalah sebutan bagi sumber arus listrik yang tidak dapat diisi ulang ketika energinya habis, contohnya seperti baterai kering dan elemen volta.Elemen sekunder adalah sebutan bagi sumber arus listrik yang dapat diisi ulang ketika energinya habis, contohnya seperti akumulator dan baterai Lithium-ion ( Li-ion ) yang digunakan pada telepon genggam atau kamera.

❐ Rumus daya listrik

Daya listrik adalah besarnya usaha yang dilakukan oleh sumber tegangan dalam 1 sekon. Jika dalam waktu t sekon sumber tegangan telah melakukan usaha sebesar W, maka daya alat tersebut, dirumuskan

[tex]P = \frac{W}{t}[/tex]

→ Keterangan :

P = daya ( Joule/sekon ) atau watt [ 1 joule/sekon = 1 watt ]W = usaha ( Joule )t = waktu ( Sekon )

❐ Rumus tegangan listrik

Tegangan listrik adalah energi potensial listrik per satuan muatan, dirumuskan

[tex]V = \frac{W}{Q}[/tex]

→ Keterangan :

V = tegangan listrik ( Volt )W = energi potensial listrik ( J )Q = muatan listrik ( C )

Dengan demikian, tegangan listrik dapat ditulis dengan rumus

[tex]V = I.R[/tex]

→ Keterangan :

V = tegangan listrik ( Volt )I = kuat arus listrik ( Ampere )R = hambatan kawat ( Ohm = Ω )

❐ Rumus hambatan listrik

Hambatan listrik dapat dipasang secara seri atau pararel. Pola pemasangan hambatan listrik dapat memengaruhi besar arus listrik yang mengalir pada suatu rangkaian listrik.

Rumus hambatan pengganti rangkaian Seri

[tex]R = R1 + R2 + R3 + dst ...[/tex]

Rumus hambatan pengganti rangkaian pararel

[tex] \frac{1}{Rp} = \frac{1}{R1} + \frac{1}{R2} \: dst...[/tex]

→ Keterangan :

Rs = hambatan total rangkaian seri (ohm atau Ω)R1 = hambatan 1 (ohm atau Ω)R1 = hambatan 1 (ohm atau Ω)R2 = hambatan 2 (ohm atau Ω)

Adapun menghitung hambatan/tahanan listrik yang mempunyai satuan Ohm yang secara umum dirumuskan

[tex]R = \frac{V}{I} [/tex]

disimpulkan bahwa besaran aliran arus sama dengan tegangan dibagi dengan hambatan, kemudian disusun sebagai rumus hambatan listrik

[tex]V = I.R[/tex]

[tex]I = \frac{V}{R} [/tex]

→ Keterangan :

R = hambatan kawat ( Ω )V = tegangan ( Volt )I = kuat arus listrik ( Ampere )

❐ Rumus kuat arus listrik

Arus listrik dapat diukur dalam satuan Coulomb/detik atau Ampere. Satuan internasional untuk arus listrik adalah Ampere ( A ). Alat ukur kuat arus listrik adalah amperemeter, yang dirumuskan

[tex] I = \frac{Q}{t} [/tex]

→ Keterangan :

I = arus listrik ( A )Q = muatan listrik ( C )t = waktu

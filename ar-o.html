2. senyawa berikut ini yang memiliki massa molekul relatif terbesar aadalah ... CO (Ar C= 12, Ar O = 16)NH3 (Ar N = 14, Ar H = 1)SO2 (Ar S = 32, Ar O =16)CO2 (Ar C = 12, Ar O = 16)SO3 (Ar S = 32, Ar O = 16)​


Jawaban:

Penjelasan:

Mr SO2 = 32 + ( 2 x 16 ) => 64

Mr CO2 = 12 + ( 2 x 16 ) => 44

Senyawa berikut ini yang memiliki massa molekul relatif terbesar adalah [tex]\sf SO_{3} \rightarrow Opsi \: E[/tex]

PENDAHULUAN

○ Nilai massa atom relatif (Ar) menunjukkan massa relatif dari 1 atom suatu unsur. Nilai Ar tiap atom bisa dilihat dalam tabel periodik.

[tex]\boxed{\bold{Ar~X~=~\frac{massa~1~atom}{\frac{1}{12}~x~massa~1~atom~C-12}}}[/tex]

○ Massa molekul relatif (Mr) = Massa relatif dari 1 molekul/senyawa. Nilainya merupakan penjumlahan dari (Ar) dari atom - atom penyusun molekul/senyawa tersebut.

[tex]\boxed{\bold{Mr~A_nB_m~=n.~Ar~A+~m.~Ar~B}}[/tex]

3. Berapakah kadar O dalam urea, CO(NH2)2? (Ar H = 1, Ar C = 12 dan Ar O = 16)​


Jawaban:

26,67%

Penjelasan:

Mr CO(NH2)2 = 60

% unsur O = ( jumlah atom O x Ar O ) / Mr urea x 100%

=> ( 1 x 16 ) / 60 x 100%

=> 26,67%

Kadar O dalam urea ([tex]\sf CO(NH_{2})_{2}[/tex]) adalah [tex]\sf 26,67\%[/tex]

PENDAHULUAN

[tex]\Large{\mathfrak{Kadar \: Unsur \: \& \: Senyawa \: dalam \: Campuran}}[/tex]

≡ Kadar Unsur dalam campuran

Contoh soal : Berapa kadar C dalam [tex]\sf CO(HN_{2})_{2}[/tex] ?

Jawab :

[tex]\sf \% C = \frac{Banyaknya \: Atom \: C \times Ar \: C}{Mr \: Senyawa} \times 100\%[/tex]

[tex]\sf \% C = \frac{1 \times 12}{60} \times 100\%[/tex]

[tex]\sf \% C = 20\%[/tex]

Contoh soal : Berapa gram besi yang terdapat dalam 200 gram [tex]\sf Fe_{2}O_{3}[/tex] ?

(Ar Fe = 56 , O = 16)

Jawab :

⇒ Mencari Mr [tex]\sf Fe_{2}O_{3}[/tex]

[tex]\sf Mr \: \: Fe_{2}O_{3}[/tex] = (2 x Ar Fe) + (3 x Ar O)

[tex]\sf Mr \: \: Fe_{2}O_{3}[/tex] = (2 x 56) + (3 x 16)

[tex]\sf Mr \: \: Fe_{2}O_{3}[/tex] = 160 gram/mol

⇒ Mencari persentase

[tex]\sf \% Fe = \frac{Banyaknya \: Atom \: Fe \times Ar \: Fe}{Mr \: Senyawa} \times 100\%[/tex]

[tex]\sf \% Fe = \frac{2 \times 56}{160} \times 100\%[/tex]

[tex]\sf \% Fe = 70\%[/tex]

⇒ Mencari Massa

[tex]\sf Massa \: Fe = \% Fe \times gr \: Fe_{2}O_{3}[/tex]

[tex]\sf Massa \: Fe = 70\% \times 200 \: gr[/tex]

[tex]\sf Massa \: Fe = 140 \: gram[/tex]

Atau ada rumus cepatnya dengan menggunakan rumus ini :

Massa A dalam m gram AxBy = [tex]\sf \frac{x \: . \: (Ar \: A)}{Mr \: A_{x}B_{y}} \times m \: gram[/tex]

[tex]\:[/tex]

≡ Kadar Senyawa dalam campuran

[tex]\sf \% Massa = \frac{Massa \: Senyawa}{Massa \: Campuran} \times 100\%[/tex]

PEMBAHASAN

Diketahui : Ar H = 1, Ar C = 12 dan Ar O = 16

Ditanya : Berapa kadar O dalam urea ([tex]\sf CO(NH_{2})_{2}[/tex]) = . . . ?

Jawab :

⇄ LANGKAH I (Menentukan Mr [tex]\sf CO(NH_{2})_{2}[/tex])

Mr [tex]\sf CO(NH_{2})_{2}[/tex] = Ar C + Ar O + (2 x Ar N) + (4 x H)

Mr [tex]\sf CO(NH_{2})_{2}[/tex] = 12 + 16 + (2 x 14) + (4 x 1)

Mr [tex]\sf CO(NH_{2})_{2}[/tex] = 28 + 14 + 4

Mr [tex]\sf CO(NH_{2})_{2}[/tex] = 60 g/mol

⇄ LANGKAH II (Menentukan persentase kadar oskigen dalam senyawa tsb)

[tex]\sf \% O = \frac{Banyaknya \: Atom \: O \times Ar \: O}{Mr \: Senyawa} \times 100\%[/tex]

[tex]\sf \% O = \frac{2 \times 16}{60} \times 100\%[/tex]

[tex]\sf \% Fe = 26,67\%[/tex]

[tex]\:[/tex]

KESIMPULAN : Jadi, Kadar O dalam urea ([tex]\sf CO(NH_{2})_{2}[/tex]) adalah [tex]\sf 26,67\%[/tex]

n = m / Mr

m = n . Mr

Keterangan :

n = molm = massaAr = massa atom relatifMr = massa molekul relatif

m = 36 gramAr H = 1Ar O = 16

n .. ?

Mr H2O = Ar H + 2 Ar O

Mr H2O = 1 + 2 . 16

Mr H2O = 1 + 32

Mr H2O = 33

- Langkah ke-2 Menentukan mol

n = m / Mr

n = 36 / 33

n = 1,09 mol

jadi, mol nya adalah 1,09 mol

28. 1. Mr CO2 = .... (diketahui Ar C = 12 dan Ar O = 16)2. Mr CH3COOH = .... (diketahui Ar C = 12, Ar H = 1 dan Ar O = 16)tolong bantu jawab dong ini penting​


Mr dari [tex] CO_2 [/tex] adalah 44 gram/mol

Mr dari [tex] CH_3COOH [/tex] adalah 60 gram/mol

»PEMBAHASAN«

[tex] \boxed{MASSA \ MOLEKUL \ RELATIF} [/tex]

merupakan massa atom relatif (Ar) dari seluruh atom penyusun suatu unsur.

Dapat ditulis:

Mr = ΣAr

keterangan:

Mr = Molekul relatif

ΣAr = Jumlah Seluruh Ar atom penyusun suatu molekul atau senyawa

===================================

Diketahui : Ar C = 12, Ar O = 16, Ar H = 1Ditanya : Mr [tex] CO_2 [/tex], Mr [tex] CH_3COOH [/tex]

Penyelesaian:

Mr [tex] CO_2 [/tex]

= (1 x Ar C) + (2 x Ar O)

= (1 x 12) + (2 x 16)

= 12 + 32

= 44 gram/mol

Mr [tex] CH_3COOH [/tex]

= (2 x Ar C) + (4 x Ar H) + (2 x Ar O)

= (2 x 12) + (4 x 1) + (2 x 16)

= 24 + 4 + 32

= 60 gram/Mol

Jadi :

Mr dari [tex] CO_2 [/tex] adalah 44 gram/mol

Mr dari [tex] CH_3COOH [/tex] adalah 60 gram/mol

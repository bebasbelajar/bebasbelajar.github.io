1. berapakah mr dari HNO3 ???


Nilai Mr HNO₃ adalah 63 g/mol. Artinya 1 mol HNO₃ memiliki massa sebesar 63 gram.

Penjelasan dengan langkah-langkah

Mr atau relative molecular weight adalah berat dari suatu molekul setiap molnya. Adapun nilai Mr dapat diketahui dengan menjumlahkan Ar dari masing-masing atom penyusunnya.

Diketahui :

Dari tabel periodik unsur diperoleh :
Ar H = 1 g/mol
Ar N = 14 g/mol
Ar O = 16 g/mol

Ditanyakan :

Mr HNO₃ = ?

Penyelesaian :

Mr HNO₃ = Ar H + Ar N + 3 x Ar O
Mr HNO₃ = 1 + 14 + 3 x 16
Mr HNO₃ = 63 g/mol

Kesimpulan :

Mr HNO₃ adalah 63 g/mol.

5. untuk membuat 800ml larutan HNO3 0,2 M dari HNO3 pekat yang kadarnya 80% dan massa jenis 1,25 gram 1ml berapa volume HNO3 pekat yang di perlukan (Mr HNO3= 63)? tolong jawab sekarang dong kak​


Pengenceran adalah proses penurunan konsentrasi zat terlarut dalam suatu larutan. Volume HNO₃ pekat yang dibutuhkan adalah 10,08 mL.

Penjelasan dengan langkah-langkah

Diketahui:

V₁ HNO₃ = 800 mL

M₁ HNO₃ = 0,2 M

% = 80%

ρ = 1,25 gram/mL

Mr HNO₃ = 63

Ditanyakan:

Volume HNO₃ pekat

Jawab:

Menentukan konsentrasi HNO₃ pekat.

M HNO₃ = (ρ x % x 10) ÷ Mr

M HNO₃ = (1,25 x 80 x 10) ÷ 63

M HNO₃ = (1000) ÷ 63

M HNO₃ = 15,873 M

Menentukan volume HNO₃ pekat.

V₁ x M₁ = V x M HNO₃ pekat

800 x 0,2 = V x 15,873

160 = 15,873 VHNO₃ pekat

V HNO₃ pekat = 160 ÷ 15,873

V HNO₃ pekat = 10,08 mL

Kemolaran menyatakan jumlah mol zat terlarut dalam tiap liter larutan

Kemolalan menyatakan jumlah zat mol terlarut dalam 1kg pelarut

fraksi mol menyatakan perbandingan antara mol zat pelarut terhadap larutan

rumus molal
m= n/p
atau
m= n×1000/p
Note
m= molalitas
p= massa pelarut (kg)
n= jumlah mol
Larutan Urea 5%

Rumus Molar
M= n/V
atau
M= n×1000/V

rumus fraksi mol
Xt = Xt/Xt+Xp
Xp = Xp/Xt+Xp

Xp+Xt = 1

soal.
hitung fraksi mol dari 10% masa larutan HNO3 dalam air dengan mr = 63

Jawab
misalkan massa = 100gram
maka massa HNO3 = 10/100×100= 10gram
maka massa air = 100%-10%/100×100= 90gram

Mencari mol kedua senyawa
n HNO3 = massa / Mr HNO3
n HNO3 = 10/63
n HNO3 = 0,158mol

n H2O = massa / Mr H2O
n H2O = 90/18
n H2O = 5mol

mencapai fraksi mol

Xt = fraksi mol HNO3
Xp = fraksi mol H2O

Xt = Xt/Xp/Xt
Xt = 0,158/5+0,158
Xt= 0,037

Xp = Xp/Xp+Xt
Xp = 5/5,158
Xp = 0,9693

21. 1. Sebanyak 0,63 gram HNO3 dilarutkan dalam 1 liter aquades. Jika Mr HNO3= 63, maka pH larutan yang terbentuk adalah


Jawaban:

pH = 2

HNO₃ → H⁺ + NO₃⁻

valensi asam = 1

n = massa/ Mr

n = 0.63/ 63 = 10⁻² mol

M = mol/ volume

M = 10⁻²/1L = 10⁻²

[H⁺]= M. valensi

[H⁺]= 10⁻² x 1 = 10⁻²

pH = - log [H⁺]

pH = - log( 10 ⁻²)

pH = 2

24. berapa ml HNO3 yg dibutuhkan dalam membuat 1000ml larutan HNO3 10%? Diketahui Mr HNO3 = 63 Massa jenis = 1.40 g/ml Kemurnian HNO3 = 65%


Diketahui:
Mr HNO3 = 63
Massa jenis = 1.40
Kemurnian HNO3 = 65%

Untuk membuat larutan dari larutan baku/lebih pekat bisa digunakan rumus pengenceran, v1. M1 = v2.M2.

Karena tidak diketahui molaritasnya, bisa dicari dengan rumus:
((rho/massa jenis) x %massa x 10) / mr

Maka untuk M1, HNO3 65%;
M1 = (1,4 x 65 x 10) / 63 = 14,444

Dan M2, HNO3 10 %
M2 = (1,4 x 10 x 10) / 63 = 2.222

mL HNO3 yang dibutuhkan;
V1. M1 = v2. M2
V1 x 14,444 = 1000 mL x 2,222
V1 = 153 (*hasil pembulatan, karena dalam prakteknya susah untuk mengambil larutan dalam bentuk desimalnya)

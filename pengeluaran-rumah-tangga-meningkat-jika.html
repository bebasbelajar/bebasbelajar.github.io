3. pengeluaran rumah tangga negara meningkat apabila


Apa rumah tangga negara yang dimaksud rumah tangga konsumen? atau pemerintah? Sedikit membingungkan namun akan saya jelaskan kedua2nya.

1. Rumah Tangga Konsumen akan meningkatkan pengeluaran jika pendapatan mereka melalui penyedia faktor produksi juga meningkat. Pengeluaran yang terus meningkat lama kelamaan akan menimbulkan inflasi.

2. Pemerintah akan meningkatkan pengeluaran jika dana alokasi anggaran kurang memadai untuk kesejahteraan masyarakat, terutama kalangan menengah ke bawah hingga kalangan miskin. Pengeluaran yang dikeluarkan dapat berupa rusun atau apartmen. Sementara di lain pihak pemerintah juga menaikkan pengeluarannya untuk memperbaiki infrastruktur. Infrastruktur dapat berupa jalan tol,rel kereta maupun bangunan pemerintah. Kemudian, yang lainnya juga dapat berupa melawan resesi yang terjadi di perekonomian. Asumsi ini dikemukakan oleh J.M Keynes mengenai pengeluaran pemerintah. Pemerintah dinilai dapat mengeluarkan dananya untuk mengkonsumsi dan kembali mendorong laju inflasi.

4. Pada pendapatan rumah tangga sebesar Rp 1.000.000,00 pengeluaran konsumsi rumah tangga sebesar Rp 1.150.000,00. Pada saat pendapatannya meningkat menjadi Rp 2.000.000,00 pengeluaran konsumsinya juga meningkat menjadi Rp 1.950.000,00.A. BEP terjadi pada pendapatan sebesar berapa ?2) pada pendapatan 875.000, berapa APS ?3) pada tingkat pendapatan berapakah tabungan rumah tangga sebesar 4% dari pendapatan dicapai ?


Pada pendapatan rumah tangga sebesar Rp 1.000.000,00 pengeluaran konsumsi rumah tangga sebesar Rp 1.150.000,00. Pada saat pendapatannya meningkat menjadi Rp 2.000.000,00 pengeluaran konsumsinya juga meningkat menjadi Rp 1.950.000,00.

1. BEP terjadi pada pendapatan sebesar 1.750.000

2) pada pendapatan 875.000,  APS adalah -0,2

3) pada tingkat pendapatan berapakah tabungan rumah tangga sebesar 4% dari pendapatan dicapai  adalah 2.187.500

Pembahasan

1. BEP terjadi pada pendapatan sebesar?

(Y - 1.000.000) / (2.000.000 - 1.000.000) = (C - 1.150.000) / (1.950.000 - 1.150.000)

(Y - 1.000.000) / 1.000.000 = (C - 1.150.000) / 800.000

(Y - 1.000.000) / 5 = (C - 1.150.000) / 4

4Y - 4.000.000 = 5C - 5.750.000

5C = 1.750.000 + 4Y

C = 350.000 + 0,8Y

1. BEP terjadi saat Y = C

     Y = C

    Y = 350.000 + 0,8Y

    0,2Y = 350.000

     Y = 1.750.000

2) pada pendapatan 875.000, berapa APS ?

    S = -350.000 + 0,2Y

    S = -350.000 + 0,2 (875.000)

       = -350.000 + 175.000

        = -175.000

    APS = S / Y

          = -175.000 / 875.000

           = -0,2

3) pada tingkat pendapatan berapakah tabungan rumah tangga sebesar 4% dari pendapatan dicapai

         S = -350.000 + 0,2Y

    4% Y = -350.000 + 0,2Y

    0,04Y - 0,2Y = -350.000

     -0,16Y = -350.000

       Y = 2.187.500

6. Pada tingkat pendapatan sebesar Rp 1.000.000 pengeluaran konsumsi rumah tangga sebesar Rp 1.300.000 ketika pendapatan meningkat menjadi Rp 1.250.000 pengeluaran konsumsi rumah tangga meningkat menjadi Rp 1.500.000 . berdasarkan informasi tersebut pada tingkat pendapatan berapa rumah tangga akan mencapai titik impas (break even point) ?


Y1 = 1.000.000
C1 = 1.300.000
Y2 = 1.250.000
C2 = 1.500.000

(Y - Y1) / (Y2 - Y1) = (C - C1) / (C2 - C1)
(Y - 1.000.000) / (1.250.000 - 1.000.000) = (C - 1.300.000) / (1.500.000 - 1.300.000)
(Y - 1.000.000) / 250.000  = (C - 1.300.000) / 200.000
(Y - 1.000.000) / 5 = (C - 1.300.000) / 4
4Y - 4.000.000 = 5C - 6.500.000
5C = 2.500.000 + 4Y
C = 500.000 + 0,8Y

Titik impas saat
Y = C
Y = 500.000 + 0,8Y
0,2Y = 500.000
Y = 2.500.000

Yaitu saat tingkat pendapatan Y = 2.500.000

Pada saat pendapatan meningkat menjadi 2.000.000 pengeluaran konsumsi juga meningkat menjadi 1.950.000 1) BEP terjadi pada pendapatan sebesar berapa ? 2) pada pendapatan 875.000, berapa APS ? 3) pada tingkat pendapatan berapakah tabungan rumah tangga sebesar 4% dari pendapatan dicapai ? TOLONG TULISKAN CARANYA JUGA


(Y - 1.000.000) / (2.000.000 - 1.000.000) = (C - 1.150.000) / (1.950.000 - 1.150.000)

(Y - 1.000.000) / 1.000.000 = (C - 1.150.000) / 800.000
(Y - 1.000.000) / 5 = (C - 1.150.000) / 4
4Y - 4.000.000 = 5C - 5.750.000
5C = 1.750.000 + 4Y
C = 350.000 + 0,8Y

1. BEP terjadi saat Y = C
      Y = C
     Y = 350.000 + 0,8Y
     0,2Y = 350.000
      Y = 1.750.000

     S = -350.000 + 0,2 (875.000)
        = -350.000 + 175.000
         = -175.000
     APS = S / Y
           = -175.000 / 875.000
            = -0,2

     4% Y = -350.000 + 0,2Y
     0,04Y - 0,2Y = -350.000

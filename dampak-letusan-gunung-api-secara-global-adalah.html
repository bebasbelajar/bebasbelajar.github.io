1. Dampak letusan gunung api secara global


Dampak letusan gunung api secara global adalah secara mikro terjadi perubahan iklim.

Pembahasan

Halo, BrainlyLovers!

Pada kesempatan kali ini kita akan bersama-sama menjawab sebuah kajian geografi dengan tema gunung api.

Gunung api merupakan fenomena permukaan bumi yang sudah wajar terjadi dan muncul karena adanya proses keluarnya magma dari lapisan astenosfer menuju kerak bumi dan kemudian membentuk lapisan baru.

Dampak letusan gunung api terbagi menjadi 2 macam yaitu dampak secara lokal dan global.

Dampak secara lokal adalah terjadinya banjir lahar panas dan lahar dingin, kebakaran hutan, menimbulkan infeksi saluran pernafasan bagi penduduk lokal dan menghanguskan hingga merusak bangunan yang ada di sekitar gunung tersebut.Dampak secara global adalah secara mikro terjadi perubahan iklim. Maksudnya terjadi perubahan iklim walaupun secara waktu tidak terus menerus terjadi. Namun dalam skala waktu tertentu terjadi perubahan. Hal ini seperti ketika gunung Krakatau meletus pada abad ke 18. Gas aerosol dan debu meluncur ke udara dengan volume yang sangat besar akhirnya menutupi atmosfer bumi. Atmosfer yang tertutup berdampak pada tertutupnya sinar matahari yang kemudian berdampak pada perubahan iklim di bawahnya.

Dampak postif dari letusan gunung berapi adalah :

Pembahasan :

Gunung berapi adalah fluida hasil saluran panas (batu dalam bentuk cair atau lava) yang berada di bawah permukaan bumi dan muncul ke permukaan bumi, termasuk hasil pengendapan akumulasi bahan dikeluarkan pada saat meletus.

Ketika terjadi letusan gunung berapi, magma yang terkandung dalam Magmar basement gunung berapi akan keluar sebagai lava. Selain aliran lava, kerusakan yang disebabkan oleh gunung berapi dapat terjadi dengan berbagai cara seperti berikut:

Letusan gunung berapi. Aliran lumpur. Abu vulkanik. Kebakaran hutan. Gas beracun. Gelombang tsunami. Gempa bumi.

Jenis jenis gunung api antara lain:

Gunung api aktif, adalah gunung api yang masih bekerja dan mengeluarkan asap, gempa, dan letusan. Gunung api tidak aktif, adalah gunung api yang tidak memiliki kegiatan erupsi.

Pembahasan

Bahasa Korea Khusus AngkaNol = Gong ( 공 ) Asli, Yeong ( 영 ) SinoSatu = Hana ( 하나 ) Asli, Il ( 일 ) SinoDua = Dul ( 둘 ) Asli, I ( 이 ) SinoTiga = Set ( 셋 ) Asli, Sam ( 삼 ) SinoEmpat = Net ( 넷 ) Asli, Sa ( 사 ) SinoLima = Daseot ( 다섯 ) Asli, O ( 오 ) SinoEnam = Yeseot ( 여섯 ) Asli, Yu' ( 육) SinoTujuh = Ilgop ( 일곱 ) Asli, Chil ( 칠 ) SinoDelapan = Yeodeol ( 여덟 ) Asli, Pal ( 팔 ) SinoSembilan = Ahop ( 아홉) Asli, Gu ( 구 ) SinoSepuluh = Yeol ( 열) Asli, Ship (십) Sino

Untuk belasan, pada angka asli korea kita hanya perlu menambahkan Awalan "Yeol", misalkan dua belas bahasa korea aslinya adalah "Yeoldul" ( 열둘 ) . Pada belasan angka sino korea, kita hanya perlu menambahkan Sib/Sip, misalkan dua belas bahasa korea sinonya adalah "sibi" (십이 ). Perbedaan penggunaan angka asli dan sino adalah bahwa angka asli digunakan untuk kegiatan sehari hari, sedangkan angka sino digunakan dalam hal-hal yang sakral dan religius dalam kepercayaan Korea.

12. ada yang tau nggak bahasa koreanya high school atau sekolah, atau sma?


Bahasa Korea Sekolah Menengah Atas (SMA) adalah 고등학교 (godeunghaggyo).

PEMBAHASAN

Di negara Korea, pada saat akan duduk di bangku SMA (Sekolah Menengah Atas) kamu dapat memilih beberapa jurusan pendidikan yang disediakan sekolah tersebut. Ada enam (6) jurusan, yaitu :

과학 고등 학교 (gwahag godeung haggyo) => SMA Sains.외국어 고등 학교 (oegug-eo godeung haggyo) => SMA bahasa asing.체육 고등 학교 (cheyug godeung haggyo) => SMA olahraga.예술 고등 학교 (yesul godeung haggyo) => SMA seni keterampilan.여자 고등 학교 (yeoja godeung haggyo) => SMA khusus wanita.남자 고등 학교 (namja godeung haggyo) => SMA khusus pria.

트럼프 대통령은 28일 중남미 출신 밀입국 이민 가족의 범람으로 남쪽 멕시코 국경이 디즈니랜드처럼 됐다고 불만을 터뜨렸다.

트럼프 대통령은 이날 폭스뉴스와의 전화인터뷰에서 “(밀입국 이민자 가족에 대한) 분리 정책을 중단한 뒤, 10배나 많은 사람이 가족과 함께 오고 있다”며 “(멕시코 국경이) 지금은 마치 디즈니랜드처럼 됐다”고 말했다.

트럼프 행정부는 불법 이민자 가족의 부모와 미성년 자녀를 격리해 수용하는 ‘가족 분리’ 정책을 펴다가 연방법원이 이를 금지토록 하고 아동학대라는 비난 여론이 일자 지난해 6월 중단했다.

트럼프 대통령은 여론에 떠밀려 가족 분리 정책을 중단하는 행정명령에 서명했으나, 이후로도 불법 이민자 축소에 이 정책이 가장 효과적이라는 입장을 보여왔다.

그는 “오바마 행정부, 부시 행정부, 그리고 우리 행정부에서 가족 분리 정책을 이용했을 때, 훨씬 적은 수의 사람들이 왔다”고 말했다. 이어 “그러나 지금은 이들을 추방하려면 정당한 절차가 필요하다”며 “변호인단이 방해하고 있다”고 주장했다.

트럼프 대통령은 중남미 출신 불법 이민자 행렬이 급증한 것은 미 경제는 호조인 가운데 이민법은 취약하기 때문이라며 이민법 강화 노력에 민주당이 동참할 것을 촉구했다. 그는 이민법 개정에 “공화당은 만장일치”라며 “(그러나 법안을 처리하려면) 민주당 표가 더 필요하다”고 말했다.

트럼프 대통령은 멕시코 국경장벽 건설을 위한 국가비상사태 선포 등 초강경 이민 정책을 내년 대선의 재선 카드로 내건 모습이다.

NBC 방송은 이달 초 복수의 정부 관계자를 인용해 “트럼프 대통령이 지난 몇 달 동안 가족 분리 정책 재개를 위해 행정부를 압박했다”는 취지로 보도했다. 그가 최근 경질한 커스텐 닐슨 국토안보부 장관은 가족 분리 정책 재개에 반대해 갈등을 빚었던 것으로 전해졌다.

트럼프 대통령은 이에 앞서 불법 이민자를 이미 ‘피난처 도시’로 이송했다고 돌발 주장을 하고 나섰다.

CNN 등은 트럼프 대통령이 지난 27일 위스콘신주 그린베이 대선 캠페인 연설에서 “10만명의 불법 이민자들이 우리 국경에 도착했고, 지역사회와 학교, 병원, 공공 자원에 전에 없던 부담을 줬다. 이제 우리는 그들 중 상당수를 피난처 도시로 보내고 있다”며 “이것이 내 대단한 구상이라고 말할 수 있어서 자랑스럽다”고 말했다고 보도했다.

피난처 도시란 트럼프 행정부의 반 이민 정책에 맞서 불법 체류자를 보호하는 LA, 뉴욕 등을 지칭한다. 모두 민주당 지지 성향이 강한 곳으로, 트럼프 대통령은 불법 이민자를 민주당 텃밭으로 몰아넣겠다고 으름장을 놔왔다.

피난처 도시 구상은 현실성이 떨어지고 위법 소지가 있다고 지적돼왔는데, 트럼프 대통령이 갑자기 이미 시행에 옮겼다고 공언한 것이다. 앞서 국토안보부가 피난처 도시 구상을 철회했다는 보도가 나왔지만 트럼프 대통령은 트위터를 통해 여전히 이 계획을 고려 중이라고 밝혔다.

21. tuliskan kata " sekolah " dalam bahasa Korea


학교 (haggyo).

Pembahasan

Sekolah (학교) adalah tempat mencari ilmu pengetahuan, setidaknya seorang pelajar berpendidikan selama 12 tahun. Didalam sekolah (학교) terdapat 5 tingkatan, yaitu :

유치원 (yuchiwon); kita mengenalnya dengan TK (Taman Kanak-Kanak) dan sama halnya dengan Kindergarten (dalam bahasa Inggris).초등학교 (Chodeung haggyo); kita mengenalnya dengan SD (Sekolah Dasar) dan sama halnya dengan Primary School (dalam bahasa Inggris).중학교 (Jung haggyo); kita mengenalnya dengan SMP (Sekolah Menengah Pertama) dan sama halnya dengan Junior High School (dalam bahasa Inggris).고등학교 (Godeung haggyo); kita mengenalnya dengan SMA (Sekolah Menengah Atas) dan sama halnya dengan High School (dalam bahasa Inggris).대학교 (Dae haggyo); kita mengenalnya dengan Universitas atau Pendidikan Perguruan Tinggi (kuliah) dan sama halnya dengan College atau University (dalam bahasa Inggris).

학교 (haggyo) termasuk 공공 장소 (gong-gong jangso) artinya “tempat umum” yaitu tempat yang memberikan pelayanan kepada masyarakat umum. Berikut kosakata terkait :

사무실 (shamushil) : Kantor (umum).역 (yeog) : Stasiun.은행 (eunhaeng) : Bank.병원 (bhyeongwon) : Rumah sakit.공항 (ghonghang) : Bandara.시장 (shijang) : Pasar.터미널 (teomineol) : Terminal Bus.공중 전화 (Ghongjhung Jheonhwa) : Fasilitas telepon publik.가게 (kage) : Toko.공원 (gongwon) : Taman.예배 장소 (yebae jangso) : Tempat Ibadah.운동장 (undongjang) : Taman bermain.슈퍼마켓 (syupeomakes) : Supermarket, Swalayan, atau Pasar Modern.식당 (sigdang) : Restoran.

27. ✨Quizz #290 ✨Apa bahasa Korea dari kalimat "Maesaroh membantu Romelah berjualan cilok goreng di sekolah".​


Bahasa Korea dari "Maesaroh membantu Romelah berjualan cilok goreng di sekolah" adalah 매사로는 로멜라가 학교에서 튀긴 실록을 판매하도록 돕습니다.

Pembahasan

Pada "Maesaroh" diganti menjadi "Maesalo", pada Romelah diganti menjadi "Lomella", di Negara Korea tidak mengenal Cilok jadi kita tulis "cilok" dalam huruf Hangeul yaitu "실록 (sillok)", Berikut adalah Penjelasannya.

- Pelajari Pola Kalimat Korea : https://brainly.co.id/tugas/28245041

Rumus-Rumus Kalimat Korea.

S - P menjadi S - P

S - P - O menjadi S - O - P

S - P - K menjadi S - K - P

S - P - O - K menjadi S - K - O - P

Contoh Kalimat.

Maesaroh membantu Romelah berjualan cilok goreng di sekolah.

- Maesaroh, Romelah : Subjek.

- Membantu, Berjualan : Predikat.

- Cilok goreng sebagai Objek

- ( di ) Sekolah sebagai Keterangan Tempat.

Pola Kalimat

S - K - O - P

매사로는 로멜라가 학교에서 튀긴 실록을 판매하도록 돕습니다. Maesalo-neun Lomella-ga haggyo-eseo twigin sillog-eul panmaehadolog dobseubnida.

• 매사로 (Maesalo) : Maesaroh Sebagai Subjek.

• 는 (neun) : Partikel Penanda Topik.

• 로멜라 (Lomella) : Romelah Sebagai Subjek.

• 가 (ga) : Partikel Penanda Subjek.

• 학교 (Haggyo) : Sekolah Sebagai Keterangan Tempat.

• 에서 (eseo) : "di" sebagai Partikel Keterangan Tempat.

• 튀긴 실록 (Twigin Sillog) : Cilok goreng Sebagai Objek.

• 을 (eul) : Partikel Penanda Objek.

• 판매하도록 (Panmaehadolog) : Jualan Sebagai Predikat.

• 돕습니다 (Dobseubnida) : Membantu Sebagai Predikat.

- Pelajari Lebih Lanjut (Abjad Korea) : https://brainly.co.id/tugas/35659774

Keterangan dari Jawaban

• Subjek disimbolkan dengan "S".

• Predikat disimbolkan dengan "P".

• Objek dilambangkan dengan "O".

• Keterangan dilambangkan dgn "K".

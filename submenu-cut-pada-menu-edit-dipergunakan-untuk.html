12. Sebutkan submenu-submenu yang ada pada menu edit beserta fungsinya


Submenu yang ada pada menu edit Ms. Word 2003:

Undo / Undo Typing : Perintah untuk membatalkan perintah terakhir.

Redo : Perintah untuk membatalkan perintah undo.

Repeat Typing : Perintah untuk mengulangi perintah terakhir.

Cut : Perintah untuk mengopi teks dengan menghapus teks yang dicopy.

Copy : Perintah untuk mengopi teks tanpa menghapus teks yang dicopy.

Office Clipboard : Perintah untuk menampilkan task pane clipboard.

Paste : Perintah untuk menyalin teks yang sudah dicopy.

Paste Spesial : Perintah untuk menyalin teks dengan berbagai macam tipe teks.

Paste as Hyperlink : Perintah untuk menyalin teks dengan bentuk teks hyperlink.

Clear : Perintah untuk menghapus teks atau gambar yang salah atau yang tidak diinginkan.

Select All : Perintah untuk memilih semua teks (yang sudah di tulis) yang terdapat di dokumen.

Find : Perintah mencari teks atau kata yang terdapat di dokumen.

Replace : Perintah untuk mengganti teks.

Go To : Perintah untuk berpindah halaman, baris, kolom, objek, footnote, endnote, bookmark, comment, table, dan lainnya.

Links : Perintah untuk membuat hubungan dengan file / dokumen lain.

Object : Perintah untuk mengedit objek.

-untuk lebih jelasnya, silakan perhatikan gambar berikut ini-

30. submenu dibawah ini adalah bagian dari menu edit


1. Undo create, yaitu untuk membatalkan sebuah perintah/langkah yang baru saja anda lakukan
2. Redo, Kembali ke yang sudah terjadi, jika anda kebabalasan klik Undo maka anda bisa kembali lagi dengan Redo
3. Repeat, yaitu untuk mengulang
4. Cut, untukmemotong
5. Copy, untuk mengkopy atau menggandakan objek
6. Paste, untuk menerapkan perintah baik copy maupun cut
7. Paste spesial, untuk menerapkan perintah cpy dengan kedaan kusus
8. Delet, untuk menghapus objek
9. Symbole, untuk edit simbol
10. Duplikate, menggandakan objek tanpa perintah copy
11. Clone, untuk menggandakan objek mengikuti objek utama
12. Copy properties from, untuk mengkopy dari propertis
13. Find and repleace, untuk mencari data dan mengganti objek
14. Insert internet objek, untuk menyisipkan objek dari website
15. Insert barcode, untuk menyisipkan barcode/kode batang
16. Insert new objek, untuk memasukkan sebuah objek baru
17.Properties, untuk menampilkan propertis sebuah objek

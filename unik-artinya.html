4. Nama file data bisnis harus unik, artinya Nama file data bisnis harus unik, artinya


Nama file data bisnis harus unik, artinya nama file harus spesifik tidak ada yang menyamai.

Pembahasan

File data bisnis adalah file yang digunakan untuk menyimpan seluruh data akuntansi yang dibuat pada program MYOB. File tersebut adalah unik, spesifik dan tunggal. File data bisnis hanya sekali dibuat dan tidak perlu dibuat lagi untuk periode-periode berikutnya.

Langkah-langkah membuat file data bisnis baru:

1. Setelah menu welcome to MYOB Accounting terbuka, klik create a new company file

2. Akan tampil form New Company File Assistant. Introduction,  Klik Next >

3. Pada windows Company Information, lengkapi data pada kolom:

- Serial number

- Company Name

- Company  No

-  Vat No

- Adrress

- Phone Number

- Fax Number

- Email Address

- Klik Next>

4. Pada windows Accounting Information, lengkapi data pada kolom:

- Current Financial Year, isikan periode tahun akuntansinya

- Last Month of Financial Year, isikan bulan terakhir periode akuntansi sebelumnya

- Conversion Month, isikan bulan yang akan kita proses atau bukukan

- Number of Periods, pilih twelve atau Thirteen jika data penyesuaian dicatat khusus

- Klik Next>

5. Periksa kembali data akuntansi yang sudah dientry, bila ada kesalahan klik <Back dan betulkan, apabila sudah benar klik Next>

6. Pada window Account List, pilihan akun yang bisa digunakan adalah :

- I would like to start with....Memilih salah satu dari account yang telah disediakan oleh MYOB

- I would like to import....Memilih import account dari file setelah selesai membuat file

- I would like to build my own....Membuat account sendiri saat menggunakan MYOB

Pilih no 2 atau 3 klik Next>

7. Pada window Company file tentukan:

- Directory dan folder serta nama file yang akan digunakan tersimpan. Secara default file akan tersimpan pada C:\Accounting 17\nama perusahaan. Apabila ingin menggantinya klik tombol Change dan tentukan Directory dan folder serta nam file yang baru

- Klik Next>

Tunggu beberapa saat untuk proses penyimpanan yang dilakukan oleh komputer

8. Pada window Conclusion, berarti pembuatan file baru sudah selesai , klik tombol Command Centre untuk masuk ke menu utama MYOB Accounting.

7. Jelaskan apa arti /makna keunikan jasmani setiap manusia dan arti /makna keunikan dalam hal kemampuan/bakat setiap orang


Keunikan jasmani setiap manusia memiliki makna: semua orang di muka bumi ini memiliki fitur dan ciri-ciri fisik yang berbeda satu sama lain. Tidak ada manusia yang terlihat persis sama seperti manusia lain. Meskipun ada dua orang yang kembar, pasti ada saja perbedaan fisik yang terdapat antara keduanya. Maka dari itu, semua orang memiliki keunikan jasmani.

Keunikan dalam kemampuan/bakat memiliki makna bahwa: semua orang memiliki sebuah ptotensi kemampuan yang berbeda-beda di dalam dirinya. Ada orang yang memiliki bakat menulis, ada yang bakat berhitung, ada yang bakat bernyanyi. Semua orang memiliki bakat yang berbeda, sehingga disebut sebagai keunikan kemampuan/bakat.

Pembahasan

Keunikan adalah sifat khusus yang dimiliki seseorang yang dapat membedakannya dengan orang lain. Keunikan jasmani dapat dilihat dari bentuk fisik seseorang, sedangkan keunikan bakat dapat diamati dari kemampuan seseorang melakukan sesuatu.

b. periode getaran = T = t/n = 1/5 = 0,2 sekon

3. jika waktu yang diperlukan untuk mengalami 3 getaran adalah 3 sekon maka berapa frekuensi getaran dan periode nya?

b. T = t/n = 3/3 = 1 sekon

4. jika waktu yang diperlukan untuk mengalami 2 getaran adalah 4 sekon maka berapa frekuensi getaran dan periode nya?

a. f = n/t = 2/4 = 0,5

b. T = t/n = 4/2 = 2 sekon

Rumus mencari frekuensi

[tex]\boxed{\boxed{f=\frac{n}{t}}}\\\boxed{\boxed{f=\frac{1}{T}}}[/tex]

Rumus mencari Periode

[tex]\boxed{\boxed{T=\frac{t}{n}}}\\\boxed{\boxed{T=\frac{1}{f}}}[/tex]

dimana:

f = frekuensi (Hz)

T = Periode (s)

n = banyak getaran/gelombang

t = waktu (s)

Diketahui:

T = 1/4 s

n = 1200 getaran

f dan t?

Jawab:

Cari frekuensi

[tex]f=\frac{1}{T} \\f=\frac{1}{\frac{1}{4}\ s }\\ \boxed{f=4\ Hz}[/tex]

Mencari waktu getaran

[tex]T=\frac{t}{n} \\t=Tn\\t=(\frac{1}{4}\ s)(1200\ getar)\\ \boxed{t=300\ s}[/tex]

PEMBAHASAN

Untuk menentukan frekuensi getaran, kita bisa menggunakan rumus berikut.

[tex] \boxed{f = \frac{n}{t}}[/tex]

Keterangan :

f = frekuensi getaran (Hertz)

n = banyak getaran

t = waktu getaran (sekon)

Untuk menentukan periode getaran, kita bisa menggunakan rumus berikut.

[tex] \boxed{T = \frac{t}{n}}[/tex]

Keterangan :

T = periode getaran (sekon)

t = waktu getaran (sekon)

n = banyak getaran

21. sebuah getaran bergetar dengan frekuensi 40Hz. periode getaran tersebut adalah


Sebuah getaran bergetar dengan frekuensi 40 Hz. Periode getaran tersebut adalah 0,025 sekon

ㅤ

Pembahasan:

Getaran merupakan gerakan bolak-balik sekitar titik kesetimbangannya. Dalam getaran ada istilah frekuensi dan periode. Frekuensi merupakan banyaknya getaran yang dapat dilakukan benda dalam waktu satu detik. Sebaliknya Periode adalah waktu yang diperlukan benda untuk melakukan satu kali getaran.

ㅤ

Frekuensi dirumuskan

[tex]\boxed{\boxed{f = \dfrac{n}{t} }}[/tex]

ㅤ

Periode dirumuskan

[tex]\boxed{\boxed{ T = \dfrac{t}{n}}}[/tex]

ㅤ

Hubungan frekuensi dan periode

[tex]\boxed{\boxed{f = \dfrac{1}{T} }}[/tex]

[tex]\boxed{\boxed{T = \dfrac{1}{f} }}[/tex]

ㅤ

Keterangan : f = frekuensi (Hz)

ㅤㅤㅤㅤ ㅤㅤT = periode (s)

ㅤㅤㅤㅤㅤㅤ n = banyak getaran

ㅤㅤㅤㅤ ㅤㅤ t = waktu (s)

ㅤ

Penyelesaian:

Diketahui : f = 40 Hz

ㅤ

Ditanyakan : T = … ?

ㅤ

Jawab :

T = [tex]\dfrac{1}{f}[/tex]

ㅤ= [tex]\dfrac{1}{40}[/tex]

ㅤ= 0,025 s

ㅤ

Jadi, periode getaran tersebut adalah 0,025 sekon

ㅤ

Penjelasan:

Kalimat pada soal di atas menunjukkan pola tenses Simple Present Tense. Simple Present Tense adalah tenses yang digunakan ketika suatu kejadian sedang berlangsung saat ini atau kejadian yang berlangsung berulang kali (kebiasaan).

Untuk menyatakan Simple Present Tense pada kalimat positif kita dapat menggunakan pola S (I, you, we, they) + V1 atau S (he, she, it) + V1 + s/es. Kata "oranges" pada soal merupakan subjek they. Dengan begitu, jika kita turunkan sesuai pola, maka bagian rumpang diisi dengan kata "contain", sehingga secara utuh kalimatnya menjadi "Oranges contain a lot of Vitamins C", sedangkan pilihan lainnya keliru secara gramatikal.

7. A box contains 56 apples and some oranges. the ratio of the number of apples to the number of oranges is 8 : 3. Find the initial number of oranges in the box.


Jawaban:

There is 21 oranges.

Penjelasan dengan langkah-langkah:

Ratio for Apples (a) is 8

Ratio for Oranges (o) is 3

Total Fruit in the Box (t) is 56

To find how many oranges in that Box, we can use this formula:

[tex] \frac{o}{a } \times t =oranges \: we \: have[/tex]

So that can be:

[tex] \frac{3}{8} \times 56 = 21[/tex]

So we have 21 oranges in that Box
